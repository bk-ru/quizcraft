    Используемые endpoint-ы

    Frontend вызывает только перечисленные маршруты backend.

    • POST/documentsЗагрузка документа и получение document_id
    • POST/documents/{document_id}/generateЗапуск генерации и получение quiz payload
    • GET/quizzes/{quiz_id}Получить сохранённый квиз для редактирования
    • PUT/quizzes/{quiz_id}Сохранить обновлённый квиз
    • GET/quizzes/{quiz_id}/export/jsonСкачать квиз в формате JSON
